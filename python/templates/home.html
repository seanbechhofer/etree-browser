About

This demo illustrates query over our Linked Data conversion of the Internet Archive's Live Music Archive (LMA) metadata. The LMA contains some 100,000 live recorded performances, recorded with the consent of the artists. Metadata describing the event is included, largely as free text.

Conversion

Metadata conversion was done by downloading the XML metadata descriptions, converting this into an RDF representation, then using some simple techniques to try and align:

  • Artists with Artists in MusicBrainz;
  • Geographical locations with locations in GeoNames (a standardised gazeteer);
  • Venues with venues in last.fm

This then allows us to query the collection using information associated with those external data sources, for example using MusicBrainz tags to find artists. The metadata is quite dirty, though, and further cleaning needs to be done. We also plan to extract further information, for example about the lineage of the recordings, which is again encoded in free text.

The hope is that this additional metadata will allow us to extract subcollections that can be used for MIR tasks.

Interesting things about the data set are:

  • Access to audio files.
  • Multiple performances by artists.
  • The presence of cover versions, which often appear in live sets.
